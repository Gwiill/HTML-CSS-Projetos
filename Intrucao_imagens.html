Trabalhando com Imagens

Imagens Vetoriais vs Bitmap

A imagem à esquerda é uma imagem **bitmap (PNG)**, composta por pixels. Se for ampliada demais, pode perder qualidade. Já a imagem à direita é uma **imagem vetorial (SVG)**, que usa fórmulas matemáticas para formar os desenhos, mantendo a qualidade mesmo quando redimensionada. Vetores são ideais para logotipos e ícones escaláveis.

Imagens com Fundo Transparente

Essas imagens demonstram o uso de **transparência**. Tanto o formato **GIF** quanto o **PNG** podem ter partes transparentes, o que é útil para sobrepor imagens em fundos diferentes. No geral, **PNG** tem melhor qualidade e suporte para mais cores, sendo mais indicado para transparências modernas.

GIF Animado

O formato **GIF** também pode ser usado para **animações simples**. Ele reproduz uma sequência de quadros, como mostrado acima, criando um pequeno vídeo em looping. Embora limitado em cores, é bastante usado em memes e ícones animados.

Float (Texto com imagem ao lado)

HTML (HyperText Markup Language) é a linguagem usada para estruturar o conteúdo de páginas na web. Ele define elementos como títulos, parágrafos, listas, links, imagens e muito mais. Sem HTML.

CSS (Cascading Style Sheets) é responsável por estilizar o conteúdo definido no HTML. Com ele, podemos mudar cores, fontes, tamanhos, posicionamentos, adicionar animações e criar layouts modernos. HTML é a estrutura; CSS é o visual.
